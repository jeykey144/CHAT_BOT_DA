Create exactly ONE valid Plotly box plot and assign it to variable: result.

## Libraries

- pandas as pd
- numpy as np

- plotly.express as px

## Data source
The datasets are already loaded as pandas DataFrames:
- DF_1, DF_2, DF_3, ...

Choose the most relevant DataFrame for the user request.

## Critical reasoning rules
1) A box plot shows the DISTRIBUTION of values. NEVER aggregate (mean/sum/count) before plotting — pass raw row-level data directly to px.box(). Plotly computes quartiles automatically.
2) NEVER do `df[cols].mean()` or any cross-column aggregation before a box plot. That reduces each column to a single scalar value, destroying the distribution.
3) For a single column distribution: `px.box(df, y='column_name')` or `px.box(df, x='group_col', y='value_col')`.
4) For wide-format data where MULTIPLE COLUMNS are the categories (e.g., toan, van, anh are separate columns): use `pd.melt()` to convert to long format first, then plot with `px.box(df_long, x='variable', y='value')`.
5) If there are too many categories, sort and keep top 10 or top 20 only when necessary.
6) Use the FULL relevant dataset unless the user explicitly asks for a subset.
7) Never use markdown fences. Output raw Python code only.
8) Do not call fig.show().

## Data cleaning & Filtering rules
Before plotting and aggregating:
- create a copy: df = DF_n.copy()
- CRITICAL FILTERING: If the user explicitly asks for a subset of data (e.g., "sản phẩm là laptop", "năm 2023", "chỉ lấy vùng A"), you MUST filter the DataFrame using pandas boolean indexing (e.g., `df = df[df['product'].str.contains('laptop', case=False, na=False)]`) BEFORE applying any groupby or calculations.
- convert required numeric columns with pd.to_numeric(..., errors="coerce")
- drop rows where required x or y values are missing
- if the chart is based on aggregated values, aggregate first into a new DataFrame named agg

## Wide-format data (multiple numeric columns as categories)
- If the user asks for distribution across "each category" where categories are MULTIPLE DISTINCT NUMERIC COLUMNS (e.g., each subject has its own column):
  - Use `pd.melt()` to reshape to long format: `df_long = df[numeric_cols].melt(var_name='category', value_name='value')`
  - Then plot: `px.box(df_long, x='category', y='value')`
  - NEVER compute `df[cols].mean()` — that destroys the distribution and produces a scalar per column.
