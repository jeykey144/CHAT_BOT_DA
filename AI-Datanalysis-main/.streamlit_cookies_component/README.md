# Streamlit Cookies Component Folder

## Muc dich

Folder nay chua frontend bundle cua custom Streamlit component duoc dung de doc va ghi cookie session.

## Nhom file trong folder

- `index.html`: entry HTML cua component
- `asset-manifest.json`: manifest cua build frontend
- `static/js/*`: cac bundle JavaScript da build

## Luu y

- Day la tai nguyen frontend da build san.
- File Python su dung folder nay nam o `ai_datanalysis/secure_cookie_manager.py`.
- Thuong chi sua khi rebuild component cookie.
